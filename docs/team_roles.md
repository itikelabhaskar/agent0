# Team Roles

## Data Engineer
- Maintains data pipelines
- Updates SQL templates
- Manages BigQuery datasets

## ML Engineer
- Develops agent logic
- Implements new tools
- Optimizes model performance

## DevOps Engineer
- Manages deployments
- Configures CI/CD pipelines
- Monitors system health

## Product Manager
- Defines requirements
- Prioritizes features
- Coordinates with stakeholders
